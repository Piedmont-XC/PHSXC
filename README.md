# PHSXC GitHub Pages Site

This is a simple mobile-friendly summer training app for Piedmont High School Cross Country.

## Live site

The site is published through the Piedmont-XC GitHub organization at:

`https://piedmont-xc.github.io/PHSXC/`

## Files

- `index.html` — main app page
- `styles.css` — visual styling
- `app.js` — workout display logic and Google Apps Script connection
- `log.html` / `log.js` — workout logging page
- `my-log.html` / `my-log.js` — individual athlete log lookup
- `leaderboard.html` / `leaderboard.js` — summer team board
- `manifest.webmanifest` — lets runners save it to their phone home screen
- `service-worker.js` — clears old cached site files

## GitHub Pages Setup

Repository owner:

`Piedmont-XC`

Repository name:

`PHSXC`

GitHub Pages should be configured as:

Settings → Pages → Build and deployment → Source: Deploy from a branch  
Branch: `main`  
Folder: `/root`  
Save

The site will publish at:

`https://piedmont-xc.github.io/PHSXC/`
